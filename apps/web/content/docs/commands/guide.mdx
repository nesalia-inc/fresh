---
title: fresh guide
description: Manage personal documentation guides
---

The `guide` command creates and manages personal documentation guides.

## Usage

```bash
fresh guide <SUB_COMMAND> [OPTIONS]
```

## Sub Commands

### create

Create a new guide:

```bash
fresh guide create my-guide --content "My guide content" --title "My Guide Title"
```

### list

List all guides:

```bash
fresh guide list
```

### show

Display a guide:

```bash
fresh guide show my-guide
```

### delete

Delete a guide:

```bash
fresh guide delete my-guide
```

### search

Search across guides:

```bash
fresh guide search "keyword"
```

## Use Cases

- Create quick reference guides
- Combine multiple documentation sources
- Add personal notes alongside documentation
- Build custom tutorials
